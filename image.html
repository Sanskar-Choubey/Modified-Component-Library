Image

The Image Component is used to display images.You can use them in your cards,background,display-image and what else.This component also contains responsive behaviour(so that they never become larger than their parent elements) and add lightweight styles to them.

Some example

Responsive Image

By responsive we mean to preserve the aspect ratio and make sure that the image doesn't come out of parent container.

Rounded Image

This component contains image in rounded shape which can be utilised like avatar by reducing size and almost everywhere based on requirement.

Aligning Image

This component will contain image in different alignment.You can do this using the FLOAT property or by using flex display.
