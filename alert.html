Alert

Alerts are used to communicate a state that effects a system,feature or page.

Some Example

Some error occured.Please check your connection

Alert Status

There are various states of the alerts.This affects the color schemes and icon used. Based on the requirements these can be used accordingly.

Success!Operations performed successfully
Warning!Your subscription will end soon.
Info!The system will go under maintainence on 30th January.
